# 公司抽奖工具

一个功能完整、界面美观的抽奖工具，专为公司活动设计，支持自定义奖项、导入人员名单、实时抽奖和中奖记录导出。

## 功能特点

- 🎨 **自定义活动**：支持设置活动名称、上传背景图
- 🏆 **多奖项管理**：可添加多个奖项，每个奖项可设置名称、名额和图片
- 📋 **人员管理**：支持手动添加或通过CSV文件导入人员名单，可设置每人的抽奖票数
- 🎲 **公平抽奖**：基于票数的概率计算，同一奖项每人最多中奖一次
- 🎉 **视觉效果**：支持全屏模式、中奖动效和彩带动画
- 📊 **中奖记录**：实时显示中奖记录，支持导出为Excel格式

## 快速开始

### 方法1：使用GitHub Pages（推荐）

1. 访问项目的GitHub Pages URL（由开发者提供）
2. 直接在浏览器中打开即可使用

### 方法2：本地运行

1. **下载项目文件**：克隆或下载本仓库到本地
2. **启动本地服务器**：
   - Windows：在项目文件夹中按住Shift键+鼠标右键，选择"在此处打开PowerShell窗口"，输入 `python -m http.server 8000`
   - Mac：打开终端，cd到项目目录，输入 `python3 -m http.server 8000`
3. **访问应用**：打开浏览器，输入 `http://localhost:8000/`

## 使用指南

### 1. 基础设置

1. **活动名称**：在左侧面板输入活动名称
2. **活动背景**：点击"选择文件"上传活动背景图

### 2. 奖项设置

1. **添加奖项**：在"奖项设置"表格中，填写奖项名称、名额，上传奖项图片
2. **管理奖项**：点击"+"按钮添加更多奖项，点击"-"按钮删除奖项

### 3. 人员设置

1. **手动添加**：在"人员设置"表格中，填写人员姓名和抽奖票数
2. **导入CSV**：点击"导入CSV"按钮，选择包含人员名单的CSV文件
   - CSV文件格式：每行包含"姓名,票数"，如 `张三,3`

### 4. 开始抽奖

1. **保存设置**：点击"保存设置"按钮，确保所有设置已生效
2. **选择奖项**：在右侧抽奖面板的"选择奖项"下拉菜单中选择要抽取的奖项
3. **开始抽奖**：点击"开始抽奖"按钮，等待滚动动画
4. **停止抽奖**：点击"停止抽奖"按钮，系统会显示中奖人员
5. **查看记录**：中奖记录会实时显示在右侧面板，可点击"导出Excel"按钮导出记录

### 5. 重置功能

- **重置抽奖**：点击"重置抽奖"按钮，清空当前奖项的中奖记录，恢复名额
- **重置设置**：点击"重置设置"按钮，清空所有设置，恢复初始状态

- **性能优化**：使用防抖处理、事件委托等技术提升性能

## 注意事项

1. **浏览器兼容性**：推荐使用Chrome、Firefox、Edge等现代浏览器
2. **数据安全**：设置和中奖记录仅保存在当前浏览器中，建议及时导出中奖记录
3. **音频播放**：抽奖过程会播放背景音乐，如不需要可在浏览器中静音
4. **全屏模式**：点击右上角的全屏按钮可进入全屏模式，获得更好的视觉效果

## 常见问题

### Q: 为什么抽奖按钮是灰色的？
A: 可能是因为当前奖项的剩余名额为0，或未选择奖项，请检查奖项设置。

### Q: 导入CSV文件失败怎么办？
A: 请确保CSV文件格式正确，每行包含"姓名,票数"，如 `张三,3`，且编码为UTF-8。

### Q: 中奖记录可以导出吗？
A: 可以，点击右侧中奖记录面板的"导出Excel"按钮，会下载一个CSV文件，可直接用Excel打开。

### Q: 如何调整抽奖动画速度？
A: 抽奖动画速度已优化，无需手动调整，确保在各种设备上都有流畅的体验。
